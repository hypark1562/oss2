# 🚨 Data Leakage Report

### 1. 개요
모델 학습 시 타겟 정보(승패)를 미리 알 수 있는 '사후 정보(Posterior Info)'를 탐지하고 제거함.

### 2. 제거된 컬럼 목록
- **gold_earned:** 게임이 끝나야 확정되는 수치이므로, 이를 통해 승패를 예측하는 것은 치팅(Cheating)에 해당함.
- **total_damage_dealt:** 상동. 게임 중 실시간 예측 모델이라면 사용 불가.

### 3. 결론
위 컬럼들을 Feature에서 제외하여 실시간 예측 환경과 동일한 조건을 구성함.
